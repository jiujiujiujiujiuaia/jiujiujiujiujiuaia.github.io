题目的一些技巧

1.当看到有重复元素的时候，可以想想是否可以先排序？因为排序会让重复元素有一个新的关系，那就是重复元素会是相邻的 2.当看到二叉搜索树，有一个性质就是，中序遍历二叉搜索的数组是一个有序数组（升序） 3.排序数组有一个特性是升序的，那么如果部分有序的数组，找到谁违法了这性质，就看是谁降序了

刷题的一些注意事项

如果自己写的代码太复杂了(行数太多，变量太多，使用的方法太tricky),要记住，要看别人的代码后，然后尝试优化，因为面试offer编程，在那种的紧张的环境中， 是不可能让你能够舒服的编码，然后完全凭肌肉记忆了。

package main
func bruthe (array []int){
	//遍历数组的左端点
	for i:=0;i<len(array);i++{
		sum :=0
		//不断向右扩展，求子数组和
		for j:=i;j<len(array);j++{
			sum +=array[j] 
        }   
    }   
}

子数组的个数

子数组的个数

假设一个数组有三个数，[1,2,3]，那么有3+2+1可能，[1,2,3,4]，那么就有4+3+2+1中可能，求和公式就是等差数列求和：(4+1) * 4 / 2 = 10

增加一个元素，增加多少个子数组

如果每增加一个新的元素，会额外增加多少种子数组呢？原来数组是[1,2,3]，现在新增加一个元素4，那么新增加的子数组就是:

[1..4] [2..4] [3..4] [4..4], 那么就是下标r-l+1

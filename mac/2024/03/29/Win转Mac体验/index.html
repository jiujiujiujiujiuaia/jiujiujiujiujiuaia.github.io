1.人生的第一台Mac

由于新工作要求使用Mac，同时自己一直听闻Mac开发起来体验很好，并且自己一直用的是苹果全家桶， 使用同一个全家桶的体验肯定很好，因此开始了自己的Mac之旅。

2.快捷键

可以参考苹果官方的一些说明，https://support.apple.com/zh-cn/HT201236

Command

Command很像Win里面的Ctrl键，因此一些常用的：

  • Command + Q 退出应用程序
  • Command + W 关闭应用程序里面的窗口
  • Command + C copy
  • Command + V paste
  • Command + -> tab 键 就是切换app
  • Command + 空格 聚焦搜索，也就是全局搜索
  • Option（或 Alt）+ Command + Esc = windows上弹出应用管理器 强制杀死不响应的进程
  • Command + Control + Q = 锁屏
  • Command + Shit 在代码软件中选中多行
  • Command + Option + D 是 程序Dock 显示和不显示
  • Fn + F 退出或者进入全屏
  • 按住Option + 方向键左右，可以实现快速跳文字（windows上是Ctrl+左右）

Terminal

  • Control + C 是 退出
  • Control + R 是 搜索

窗口操作

macOS 下对窗口的操作有 4 种，分别是关闭、最小化、全屏、最大化。

  • 「关闭」点按窗口左上角的红色关闭按钮，或者按下 Command-W。
  • 「最小化」点按窗口左上角的黄色最小化按钮 ￼，或者按下 Command-H。
  • 「全屏」点按应用窗口左上角的绿色按钮，或者按下 Control-Command-F。
  • 「最大化而不是全屏幕」双击应用顶部
  • 「切换窗口」是Command + Tab
  • 「切换tab」是Control + Tab => 网页切换和程序切换的意思

Command H 和Command M的区别

  • Command+M一个窗口或者应用是把它折叠到右下角
  • Command+H把一个窗口或者应用缩小，然后Command+Tab中可以找到

空格

对任意文件选中并且空格，就可以提前预览这个文件

3.手势

打开设置=>点击触摸板=>可以看到有哪些手势以及已经相应的动画，下面是一些推荐的手势

单指

  • 单个指头按下就是选中，可以选中，文本块儿等等
  • 单指双击就是常见的打开软件，文件，图片等等

双指

  • 开启”轻点来点按”
  • 双指轻点，就会达到windows右键的效果，例如打开一些选项
  • 如果没有开启下面的设置，那么就是双指重击可以达到一样的效果

三指

  • 打开三指拖移的选项
  • 三指就是win鼠标按住拖拽
  • 三指在文本框中选择就是快速选中文字
  • question：大面积拖拽如何解决呢？
    • 比如从一个屏幕拖拽到另外一个屏幕，受限于触摸板的面积，不好操作
    • 同时，选中超过一个屏幕的文字，受限于触摸板的面积，也不好操作

四指

  • 四指往上就是所有app展示
  • 四指左右滑是全屏应用切换，注意，应用得点击了左上角绿色折叠按钮才是全屏应用
  • 四指+大拇指分开就是显示桌面
  • 四指+大拇指合拢就是出现启动台（不常见）

4.Magic Mouse

用了一段时间手势，的确很好用，但是感觉在固定场所办公，magic mouse的效率会更高一些， 并且更加适合我这种从windows转过来的用户，更加的简单

  • 首先把鼠标调整到最灵敏的状态
  • 手势：大拇指抵住鼠标左侧，食指和中指分别是鼠标的左右两半
  • 单指上下滚动是翻页
  • 单指快速轻点两下是放大缩小
  • 单指在浏览器中左右横扫是页面前后切换
  • 双指左右横扫是全屏网页切换
  • 双指快速两下tab 是查看桌面

Anki + dicToAnki插件

  • 官网下载qt5版本而不是qt6版本（反正我自己觉得没什么区别）
  • 打开anki，登陆你的anki账号，然后下载插件1284759083
  • 选择dicToAnki
  • 登陆你的网易词典 or 欧陆词典
  • 在插件的设置中选择下载英音或者美音的音频（好像只能二选一）
  • Note!!!!:
    • 1.找到插件的代码目录，/Users/{Your_User_Name}/Library/Application Support/Anki2/addons21/1284759083
    • 2.找到worker.py, 在代码文件第一行加入import os
    • 3.找到worker.py, 把148行代码修改成下面这两行， target_directory = ‘/Users/{Your_User_Name}/Library/Application Support/Anki2/User 1/collection.media’ with open(os.path.join(target_directory, fileName), ‘wb’) as f:

然后重启anki，目的是让插件把音频下载到collection.media这个目录下，这样就可以顺利的使用这个插件了。

Note：有任何问题，多去看看Anki插件的日志，甚至加一些代码进行调试

5.Windows常见概念在Mac上替代

安装应用

除了用命令行brew install xx安装外，还可以去软件官网下载dmg后缀的文件，第一次弹出了这个框一直不理解是什么意思。

以snipaste举例子，就是让你拖动左边图标，拽到Applications这个文件夹的意思，这样“聚焦搜索”可以识别到它

应用管理器

我们常用应用管理器杀死进程或者查看CPU memory的使用状况，在Mac上也有这么一款软件

Command + 空格 => 活动监视器 => Pin到程序坞

App Data

对于App data存放在哪里，这里有一个映射关系：

  • Windows
    • C:/Users/{你的用户名}/AppData/Roaming/Anki2
  • Linux
    • /Users/{Your_User_Name}/Library/Application Support/Anki2/

命令行打开文件夹

  • Windows
    • ii . 快速打开文件夹
  • Linux
    • open .

删除应用

通常，苹果电脑卸载应用程序可以直接在Finder中实现。在Finder中查找Application，找到要卸载的app，将其拖拽到垃圾桶中。

但是，多数时候仅仅将程序移到垃圾筒是远远不够的，程序会留下preference和application support文件：

  • Preference文件是方便用户再次安装，他们通常只占很小空间。
  • application support （程序支持文件）占的空间就比较大，从几KB到甚至几GB不等，取决于程序类型。这类文件可以在系统根目录下的“/资源库/Application Support/”查得到。

也可以下载Dr. Cleaner

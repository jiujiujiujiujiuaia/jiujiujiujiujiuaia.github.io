## 人生的第一台Mac

由于新工作要求使用Mac，同时自己一直听闻Mac开发起来体验很好，并且自己一直用的是苹果全家桶，
使用同一个全家桶的体验肯定很好，因此开始了自己的Mac之旅。

## 快捷键

### Command

Command很像Win里面的Ctrl键，因此一些常用的：
* Command + C copy
* Command + V paste
* Command + ->键 就是切换app
* Command + 空格 聚焦搜索，也就是全局搜索

### 空格

对任意文件选中并且空格，就可以提前预览这个文件

## 手势

* 三指往上就是所有app
* 两个指头按下就是鼠标右键
* 单个指头按下就是选中

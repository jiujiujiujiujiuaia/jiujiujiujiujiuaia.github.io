## 2.快捷键

### Command

Command很像Win里面的Ctrl键，因此一些常用的：

* Command + Q 退出应用程序
* Command + W 关闭应用程序里面的窗口
* Command + C copy
* Command + V paste
* Command + -> tab 键 就是切换app
* Command + 空格 聚焦搜索，也就是全局搜索
* Option（或 Alt）+ Command + Esc = windows上弹出应用管理器 强制杀死不响应的进程
* Command + Control + Q = 锁屏
* Command + Shit 在代码软件中选中多行


### 窗口操作

macOS 下对窗口的操作有 4 种，分别是关闭、最小化、全屏、最大化。

* 「关闭」点按窗口左上角的红色关闭按钮，或者按下 Command-W。
* 「最小化」点按窗口左上角的黄色最小化按钮 ￼，或者按下 Command-H。
* 「全屏」点按应用窗口左上角的绿色按钮，或者按下 Control-Command-F。
* 「最大化而不是全屏幕」双击应用顶部
* 「切换窗口」是Command + Tab
* 「切换tab」是Control + Tab => 网页切换和程序切换的意思

### 空格

对任意文件选中并且空格，就可以提前预览这个文件

## 3.手势

打开设置=>点击触摸板=>可以看到有哪些手势以及已经相应的动画，下面是一些推荐的手势

### 单指

* 单个指头按下就是选中，可以选中，文本块儿等等
* 单指双击就是常见的打开软件，文件，图片等等

### 双指

* 开启"轻点来点按"
* 双指轻点，就会达到windows右键的效果，例如打开一些选项
* 如果没有开启下面的设置，那么就是双指重击可以达到一样的效果

![img_1.png](https://raw.githubusercontent.com/jiujiujiujiujiuaia/jiujiujiujiujiuaia.github.io/master/_posts/pic/winmac/20240408160859/img_1.png)

### 三指

* 三指往上就是所有app展示
* 三指左右滑是**全屏应用切换**，注意，应用得点击了左上角绿色折叠按钮才是**全屏应用** 
* 三指+大拇指分开就是显示桌面
* 三指+大拇指合拢就是出现启动台（不常见）

### Anki + dicToAnki插件

- [官网下载qt5](https://apps.ankiweb.net/)版本而不是qt6版本（反正我自己觉得没什么区别）
- 打开anki，登陆你的anki账号，然后下载插件1284759083
- 选择dicToAnki
- 登陆你的网易词典 or 欧陆词典
- 在插件的设置中选择下载英音或者美音的音频（好像只能二选一）
- **Note!!!!**:
  - 1.找到插件的代码目录，/Users/{Your_User_Name}/Library/Application Support/Anki2/addons21/1284759083
  - 2.找到worker.py, 在代码文件第一行加入import os
  - 3.找到worker.py, 把148行代码修改成下面这两行，
    target_directory = '/Users/{Your_User_Name}/Library/Application Support/Anki2/User 1/collection.media'
    with open(os.path.join(target_directory, fileName), 'wb') as f:

然后重启anki，目的是让插件把音频下载到collection.media这个目录下，这样就可以顺利的使用这个插件了。

**Note**：有任何问题，多去看看Anki插件的日志，甚至加一些代码进行调试

## 5.Windows常见概念在Mac上替代

### 应用管理器

我们常用应用管理器杀死进程或者查看CPU memory的使用状况，在Mac上也有这么一款软件

Command + 空格 => 活动监视器 => Pin到程序坞

### App Data

对于App data存放在哪里，这里有一个映射关系：
* Windows
  * C:/Users/{你的用户名}/AppData/Roaming/Anki2
* Linux
  * /Users/{Your_User_Name}/Library/Application Support/Anki2/

### 命令行打开文件夹

* Windows
  * ii . 快速打开文件夹
* Linux
  * open .

### 删除应用

通常，苹果电脑卸载应用程序可以直接在Finder中实现。在Finder中查找Application，找到要卸载的app，将其拖拽到垃圾桶中。

但是，多数时候仅仅将程序移到垃圾筒是远远不够的，程序会留下preference和application support文件：
* Preference文件是方便用户再次安装，他们通常只占很小空间。
* application support （程序支持文件）占的空间就比较大，从几KB到甚至几GB不等，取决于程序类型。这类文件可以在系统根目录下的“/资源库/Application Support/”查得到。

也可以下载[Dr. Cleaner](https://apps.apple.com/cn/app/id921458519?mt=12)
